Mini Compiler for Language "MiniL"

The aim of this project is to develop a mini-compiler for a language called "MiniL". This compiler will undergo various compilation phases including lexical analysis using the FLEX tool, syntactic and semantic analysis, and the generation of intermediate code using the BISON tool. Additionally, it will handle parallel treatments such as managing the symbol table and handling various compilation errors.
